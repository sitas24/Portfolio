ROBOTICS WHATTTTT!!!

My viewpoint of robotics was completely skewed when I learned about the arduino board. The arduino board makes it much easier for people just starting off with robots to code easily and fix all the bugs in their code in a matter of minutes by just uploading it on the arduino board. Just the thought of being able to code a robot directly from my home computer excites me. With evolving innovations, robotics can function without humans, being able to do tedious tasks so humans can be even more innovative.

What worked the best for my group was talking about our code and just bouncing ideas off of each other. The only tweak I would like in my group is if we worked faster and more efficiently.
